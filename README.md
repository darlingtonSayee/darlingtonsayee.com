# Darlington Sayee Portfolio

Static portfolio website for Darlington Sayee.

## Pages

- `index.html` - homepage
- `about.html` - professional portfolio
- `services.html` - finance, bookkeeping, QuickBooks, and social media management services
- `blog.html` - insights page
- `contact.html` - contact form and direct contact links

## GitHub Pages domain

This site includes a `CNAME` file for:

```text
darlingtonsayee.com
```

After pushing to GitHub, enable GitHub Pages from the repository settings and point the domain DNS records to GitHub Pages.
